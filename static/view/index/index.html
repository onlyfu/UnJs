UnJs 一个Single Page开发框架

UnJs特点

  • 1. 轻封装 简单易用
  • 2. 基于NodeJs的构建服务 自动模板解释，合并，压缩
  • 3. 跨域调试 可根据API情况自定义测试AJAX请求服务
  • 4. 组件模块 UI功能组件化，方便重用与维护
  • 5. 动态加载 减少基础文件大小，按需加载文件

使用

// 默认端口3000
node app.js
// 指定端口启动
node app.js -p 3100
// 指定请求服务,uat为控制器名，见根目录下controller目录
node app.js -h uat
// 启动同时编译
node app.js -b true
